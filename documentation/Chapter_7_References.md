# Chapter 7 – References

This chapter lists the external libraries, resources, and documentation used in the development of **XP Store**.

## 7.1 Libraries & Functionality

### 7.1.1 Core Framework & Backend
*   **Next.js 15 (App Router)**: The main React framework used for both frontend and backend API routes.
    *   *Source*: [https://nextjs.org/docs](https://nextjs.org/docs)
*   **MongoDB & Mongoose**: utilized for the NoSQL database and object modeling.
    *   *Source*: [https://mongoosejs.com/docs/](https://mongoosejs.com/docs/)
*   **NextAuth.js (v5 Beta)**: Used for secure authentication (Google OAuth & Credentials).
    *   *Source*: [https://authjs.dev/](https://authjs.dev/)
*   **Bcrypt.js**: For hashing and securing user passwords.
    *   *Source*: [https://www.npmjs.com/package/bcryptjs](https://www.npmjs.com/package/bcryptjs)

### 7.1.2 UI Components & Animations
Diverse sources were used to enhance the visual appeal and interactivity of the application.

*   **Uiverse.io**: A community-driven library for open-source UI elements. The following custom components were adapted:
    *   `CyberButton`: A futuristic, glitch-styled button.
    *   `NeonButton`: A glowing action button.
    *   `GlitchButton`: Used for high-impact actions.
    *   `PremiumButton`: Used for the "Back to Consoles" navigation.
    *   `WishlistButton`: Heart icon interaction.
    *   `MagicCard`: A card component with a spotlight hover effect.
    *   *Source*: [https://uiverse.io/](https://uiverse.io/)
*   **Anime.js**: A lightweight JavaScript animation library used for complex entrance animations.
    *   *Source*: [https://animejs.com/](https://animejs.com/)
*   **Cobe**: A WebGL library used to render the interactive 3D globe on the homepage.
    *   *Source*: [https://github.com/shuding/cobe](https://github.com/shuding/cobe)
*   **Radix UI**: Headless UI primitives used for accessible components like Dialogs and Dropdowns.
    *   *Source*: [https://www.radix-ui.com/](https://www.radix-ui.com/)
*   **Lucide React**: The primary icon set used throughout the application.
    *   *Source*: [https://lucide.dev/](https://lucide.dev/)
*   **Remix Icon**: Supplementary icons loaded via CDN.
    *   *Source*: [https://remixicon.com/](https://remixicon.com/)

### 7.1.3 Styling & Typography
*   **Tailwind CSS (v4)**: A utility-first CSS framework for rapid UI development.
    *   *Source*: [https://tailwindcss.com/](https://tailwindcss.com/)
*   **Geist Font**: The Vercel Geist and Geist Mono fonts were used for a modern, technical aesthetic.
    *   *Source*: [https://vercel.com/font](https://vercel.com/font)
*   **Tailwind Merge & CLSX**: Utilities for conditionally merging Tailwind class names.

## 7.2 Development Tools
*   **Visual Studio Code**: The primary Integrated Development Environment (IDE).
*   **Postman/Thunder Client**: Used for testing API endpoints during backend development.
*   **Git & GitHub**: Used for version control and source code management.
*   **Vercel**: The platform used for deployment and hosting (conceptual/if effective).

## 7.3 Design Inspiration
*   **Dribbble & Pinterest**: References for modern "Dark Mode" gaming e-commerce aesthetics.
*   **PlayStation/Xbox Official Stores**: Analyzed for layout structure and navigational flow.
